Anatomia de uma Requisição HTTP

Requisição (Request)

GET /api/users HTTP/1.1
Host: exemplo.com
User-Agent: Mozilla/5.0
Accept: application/json
Connection: keep-alive

Resposta (Response)

HTTP/1.1 200 OK
Content-Type: application/json
Content-Length: 126

{
  "users": [
    {"id": 1, "name": "João"}
  ]
}
